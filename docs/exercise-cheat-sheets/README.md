## How to use these

1. **Try the exercise first** — the goal is to learn by doing
2. **If you're stuck**, check the cheat sheet for the specific step you're on
3. **If you're in a workshop and running behind**, copy the full solution to catch up
4. **After finishing**, compare your solution with the cheat sheet — there's often more than one valid approach

Each cheat sheet includes:
- ✅ Complete, working TypeScript code
- ✅ The `server.ts` registration line
- ✅ "What to notice" callouts highlighting key MCP patterns
